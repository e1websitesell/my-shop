  • name_bn, name_en — বাধ্যতামূলক
  • basePrice — বাধ্যতামূলক, শুধু সংখ্যা
  • category — বাধ্যতামূলক, "ক্যাটাগরি" পেজে আগে বানানো স্লাগ হতে হবে
  • description_bn, description_en — অপশনাল
  • discountPrice — অপশনাল, সংখ্যা
  • tags — অপশনাল, একাধিক হলে ; দিয়ে আলাদা করুন
  • images — অপশনাল, একাধিক লিংক হলে ; দিয়ে আলাদা করুন
  • variants — অপশনাল। ফরম্যাট: নাম:কোড:দাম:স্টক, একাধিক ভ্যারিয়েন্ট | দিয়ে আলাদা করুন। যেমন: 50ml:JIRO-001-50:1200:yes|100ml:JIRO-001-100:2000:yes
